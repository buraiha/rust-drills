# Day15: ジェネリクス

**対象crate:** `day15_generics_stack`

このドリルは「**動かす → 読む → 改造 → 壊して直す → テスト → コミット**」の反復で身につけます。

## タイムボックス（60分）

- 0〜5分：ウォームアップ（fmt/clippy/test）
- 5〜15分：読む（今日のポイントを説明できるようにする）
- 15〜45分：改造ドリル（1〜2個）
- 45〜55分：壊して直すドリル（1個）
- 55〜60分：メモ＆コミット

## 実行コマンド

```bash
./rust.sh cargo test -p day15_generics_stack
./rust.sh cargo run  -p day15_generics_stack
# 仕上げ
./rust.sh cargo fmt
./rust.sh cargo clippy -p day15_generics_stack -- -D warnings
```
